Insert a semicolon at the end of each of the following lines.
-------------------------------------------------------------

var foo = 1
var bar = a'
var foobar = foo + bar

After the change the lines should look like the following

var foo = 1;
var bar = a';
var foobar = foo + bar;

Solution: Using visual block mode (Ctrl-V) select the three lines.
After that hit 'A;', then hit <Esc> to return to normal mode.
